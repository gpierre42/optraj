def optimisation.SolTemp.SolTemp.genSolTempFromStatTable (   system,
  statTable 
)
static

génère une SolTemp "aléatoirement" en se basant sur une StatTable déjà construite 

   La solution générée sera aléatoire mais va essayer de construire une solution en concordance avec la StatTable associée.
   L'idée est de générer une SolTemp ressemblant à celles que l'on aurait préalablement construite mais en introduisant
   de petits changement de répartition dans l'espoir d'en trouver une plus optimale.


       system le Systeme pour lequel on génère la solution
       statTable la StatTable (table de statistique d'affectations) sur laquelle on se base
Renvoie
Une SolTemp
